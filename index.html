हमारा दर्शन

2020 में स्थापित, MinimalLux का जन्म ऐसे कालातीत टुकड़े बनाने की इच्छा से हुआ था जो रुझानों से परे हैं। हम मात्रा से अधिक गुणवत्ता में विश्वास करते हैं, प्रत्येक वस्तु को वर्षों तक चलने के लिए डिज़ाइन करते हैं।

हमारी सामग्री नैतिक आपूर्तिकर्ताओं से सावधानीपूर्वक प्राप्त की जाती है, और प्रत्येक टुकड़े को विस्तार पर ध्यान देने के साथ तैयार किया जाता है। कम ज्यादा है - यही हमारा मंत्र है।

हम स्थायी प्रथाओं और पारदर्शी उत्पादन प्रक्रियाओं के लिए प्रतिबद्ध हैं। हमारा लक्ष्य कचरे को कम करना और ऐसे उत्पाद बनाना है जिन्हें आप वर्षों तक संजोएंगे।

हमारी कहानी

GitHub पर फ्री वेबसाइट कैसे बनाएं

चरण 1: GitHub अकाउंट बनाएं

GitHub.com पर जाएं और "Sign up" बटन पर क्लिक करें

अपना यूजरनेम, ईमेल और पासवर्ड डालें

चरण 2: नई Repository बनाएं

GitHub डैशबोर्ड पर, "New" बटन पर क्लिक करें

Repository नाम डालें (जैसे: my-website)

सुनिश्चित करें कि repository "Public" है

चरण 3: कोड अपलोड करें

Repository में जाएं और "Add file" > "Create new file" चुनें

फ़ाइल का नाम "index.html" रखें

इस पेज का सारा कोड कॉपी करके यहाँ पेस्ट करें

चरण 4: GitHub Pages सक्षम करें

Repository में "Settings" टैब पर जाएं

बाईं ओर "Pages" पर क्लिक करें

"Branch" के अंतर्गत "main" चुनें और "/ (root)" फोल्डर चुनें

"Save" पर क्लिक करें

चरण 5: अपनी वेबसाइट देखें

कुछ मिनट बाद, आपकी वेबसाइट इस लिंक पर लाइव हो जाएगी:

https://<your-username>.github.io/<repository-name>
